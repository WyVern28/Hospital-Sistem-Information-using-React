# Frontend Anahita Hospital

Antarmuka React + TypeScript untuk portal pasien dan dokter Anahita Hospital.

## Menjalankan

```bash
npm install
npm run dev
```

Secara default frontend memakai API di `http://localhost:5000`. Alamat ini dapat diubah melalui `VITE_API_URL`.

Perintah pemeriksaan:

```bash
npm run lint
npm run build
```

Dokumentasi lengkap, akun demo, dan cara menjalankan seluruh stack tersedia di [README utama](../README.md).
